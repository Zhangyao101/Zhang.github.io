Hexo
基于Hexo+Github搭建个人博客
2023-08-22
准备工作：
  1. nodejs：用来生成静态页面，下载电脑对应版本，Nodejs官网。
  2. git：安装git，将本地的hexo内容提交到github上去参考,安装git。
  3. github：拥有一个自己的github账号。
安装Hexo：
C:\Program Files\nodejs>npm install -g hexo
初始化：
PS：创建一个空的文件夹之后再执行命令初始化
D:\blog>hexo init
D:\blog> npm install
added 1 package, and audited 238 packages in 3s
27 packages are looking for funding
run npm fund for details
found 0 vulnerabilities
D:\blog>hexo s
INFO  Validating config
INFO  Start processing
INFO  Hexo is running at http://localhost:4000/ . Press Ctrl+C to stop.
尝试访问

开始关联Github：
创建github仓库 (固定格式：name.github.io)

添加git配置：
deploy:
  type: git
  repository: https://github.com/Zhangyao101/Zhangyao101.github.io.git
  branch: master
生成静态页面：
hexo generate        或者：hexo g
部署：
hexo deploy            或者：hexo d
PS:如果报错（ERROR Deployer not found: git）
npm install hexo-deployer-git --save  #安装hexo-deployer-git即可

更换主题：

Hexo主题官网,找到自己喜欢的主题，根据主题作者的Github主页配置即可。
obsidian为示例

D:\blog>git clone https://github.com/bennyxguo/hexo-theme-obsidian.git
#将blog目录下_config.yml里theme的名称修改为自己想要的主题名称
hexo clean           //清除缓存文件 (db.json) 和已生成的静态文件
hexo g             //生成缓存和静态文件
hexo d             //重新部署到服务器
